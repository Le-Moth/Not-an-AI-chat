## Список технологий и библиотек:

**Язык:** `Python 3.13` использовался для полного создания логики и бэкэнда, для **База данных:** `SqlAlchemy`, `Flask` для **Бэкенд сайта**, `OpenAI` для **Мозг ИИ** 

## 🤖 Установка и запуск:



Скачать PyCharm, затем скачать python версии 3.13, закинуть все файлы в 1 проект, открыть папку в PyCharm, получить в Groq API ключ API и вставить его в 7 строчку файла `main.py`"api_key=".

## Схема базы данных



| Поле | Тип                 | Описание                                              |
| ---- | ------------------- | ----------------------------------------------------- |
| id   | INTEGER PRIMARY KEY | id сообщения(для строения хронологии сообщений)       |
| Text | TEXT                | Текст                                                 |
| Role | TEXT                | для понимания ИИ'шкой кто пишет(пользователь или бот) |
